# CodeAlpha Student Grade Tracker

This is a beginner Java project for tracking student grades.

## Structure

- `src/Task1/Student.java` - student data model.
- `src/Task1/GradeManager.java` - grade calculation and reporting logic.
- `src/Task1/GradeTrackerApp.java` - console application entry point.

## Run

Compile:
```cmd
javac src\Task1\*.java
```
Run:
```cmd
java -cp src Task1.GradeTrackerApp
```
